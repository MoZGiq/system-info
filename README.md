# 🖥 System Info

Программа для сбора технических характеристик компьютера с согласия пользователя.
Создаёт красивый HTML-отчёт и сохраняет его на Рабочий стол.

## 📋 Возможности

- ✅ Сбор информации о CPU, RAM, GPU, дисках, сети
- ✅ Информация о материнской плате и BIOS
- ✅ Параметры дисплея и операционной системы
- ✅ Версия Java и параметры JVM
- ✅ Красивый HTML-отчёт с современным дизайном
- ✅ Полная конфиденциальность - данные не отправляются в интернет
- ✅ Согласие пользователя обязательно

## 📦 Используемые библиотеки

- [OSHI](https://github.com/oshi/oshi) — сбор системной информации
- [SLF4J](https://www.slf4j.org/) — логирование

## 🛡 Безопасность

- 🔒 **Программа не отправляет данные в интернет**
- 🔒 **Работает полностью локально**
- 🔒 **Открытый исходный код**
- 🔒 **Сохраняет отчёт только на Рабочий стол**
- 🔒 **Не модифицирует системные файлы**
